الطريقة 1️⃣: ربط GitHub مع Netlify (موصى بها - نشر تلقائي)
───────────────────────────────────────────────────────────────────

1. اذهب إلى: https://app.netlify.com

2. سجل دخول أو أنشئ حساب (يمكنك تسجيل الدخول بحساب GitHub)

3. اضغط "Add new site" → "Import an existing project"

4. اختر "Deploy with GitHub"

5. ابحث عن "SuondTube" واختره

6. إعدادات البناء (ستكون تلقائية من netlify.toml):
   ✅ Build command: npm run build
   ✅ Publish directory: dist
   ✅ Base directory: (اتركه فارغاً)

7. اضغط "Deploy SuondTube"

8. انتظر 1-2 دقيقة... ✅ تم!

9. 🎉 موقعك الآن على الهواء!
   سيكون الرابط مثل: https://suondtube.netlify.app


مميزات هذه الطريقة:
─────────────────────
✅ كل مرة تعمل push على GitHub → يتم النشر تلقائياً!
✅ رابط HTTPS مجاني
✅ CDN عالمي (سريع في كل العالم)
✅ شهادة SSL تلقائية
✅ يمكنك ربط domain مخصص لاحقاً


الطريقة 2️⃣: Netlify Drop (سريعة - للتجربة)
───────────────────────────────────────────────────────────────────

1. المشروع مبني مسبقاً (مجلد dist موجود)

2. اذهب إلى: https://app.netlify.com/drop

3. اسحب مجلد "dist" إلى الصفحة

4. انتظر ثوانٍ... ✅ تم!

ملاحظة: هذه الطريقة لن تُحدّث تلقائياً عند عمل push

═══════════════════════════════════════════════════════════════════

تغيير اسم المشروع في الملفات:
─────────────────────────────────

إذا أردت تغيير "YouTube Audio Podcast Player" إلى "SuondTube":

1. افتح package.json وغيّر:
   "name": "soundtube"

2. افتح public/manifest.json وغيّر:
   "name": "SuondTube",
   "short_name": "SuondTube"

3. افتح index.html وغيّر العنوان:
   <title>🎵 SuondTube</title>

4. احفظ التغييرات:
   ```powershell
   git add .
   git commit -m "Update branding to SuondTube"
   git push
   ```


تخصيص الأيقونات:
─────────────────────────────────

1. صمم أيقونة 512x512 بكسل

2. اذهب إلى: https://www.pwabuilder.com/imageGenerator

3. ارفع الأيقونة وحمّل الملف المضغوط

4. استبدل محتويات مجلد public/icons/

5. أعد البناء والرفع:
   ```powershell
   npm run build
   git add .
   git commit -m "Update app icons"
   git push
   ```

    💡 نصائح مهمة

✅ HTTPS إجباري لـ PWA:
   - Netlify يوفر HTTPS تلقائياً ومجاناً
   - لا تقلق بشأن SSL

✅ رابط مخصص (Custom Domain):
   - يمكنك ربط domain خاص بك من إعدادات Netlify
   - مثال: soundtube.com بدلاً من soundtube.netlify.app

✅ متغيرات البيئة:
   - إذا احتجت API keys لاحقاً
   - استخدم Environment Variables في Netlify

✅ النطاق الترددي:
   - Netlify Free: 100GB/شهر
   - كافٍ لآلاف الزيارات!
