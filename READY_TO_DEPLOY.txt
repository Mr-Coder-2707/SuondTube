   الطريقة 1: ربط GitHub (موصى بها)
   ─────────────────────────────────────
   
   1. اذهب إلى → https://app.netlify.com
   
   2. اضغط "Add new site" → "Import an existing project"
   
   3. اختر "Deploy with GitHub"
   
   4. ابحث عن "SuondTube" واختره
   
   5. اضغط "Deploy site"
   
   6. ✅ تم! الموقع على الهواء
   
   ✨ كل push على GitHub = نشر تلقائي!


   الطريقة 2: Netlify Drop (سريعة)
   ─────────────────────────────────────
   
   1. اذهب إلى → https://app.netlify.com/drop
   
   2. اسحب مجلد "dist" إلى الصفحة
   
   3. ✅ تم! الموقع على الهواء فوراً
