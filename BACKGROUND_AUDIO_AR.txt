✨ الميزات الجديدة:

1️⃣ تشغيل الصوت عند غلق الشاشة 📱
   - شغّل أي مقطع
   - اضغط زر الطاقة لإطفاء الشاشة
   - الصوت يستمر! ✅

2️⃣ التحكم من شاشة القفل 🔒
   - شغّل الشاشة (بدون فتح القفل)
   - سترى أزرار التحكم
   - Play/Pause/Next/Previous

3️⃣ التشغيل في الخلفية 🔄
   - اخرج من المتصفح
   - افتح WhatsApp أو Instagram
   - الصوت يستمر في الخلفية!

4️⃣ إشعارات ذكية 🔔
   - صورة المقطع
   - عنوان المقطع
   - أزرار التحكم

5️⃣ منع نوم الشاشة ⏰
   - الشاشة لا تنام أثناء التشغيل
   - توفير البطارية عند الإيقاف
   - تفعيل تلقائي عند العودة

🔧 التفاصيل التقنية:

✅ Media Session API
   - التحكم من شاشة القفل
   - الإشعارات الذكية
   - عرض معلومات المقطع

✅ Wake Lock API
   - منع نوم الشاشة
   - تفعيل تلقائي
   - توفير البطارية

✅ Background Audio
   - استمرار التشغيل
   - إدارة ذكية للموارد
   - تنظيف تلقائي

💡 نصائح للأداء الأفضل:

1. ثبّت التطبيق كـ PWA
   ┌───────────────────────────────┐
   │ Chrome Menu → Add to Home     │
   │ Safari Share → Add to Home    │
   └───────────────────────────────┘

2. استخدم HTTPS
   ┌───────────────────────────────┐
   │ ✅ GitHub Pages (مفعّل تلقائياً)│
   │ ✅ Netlify (مفعّل تلقائياً)    │
   └───────────────────────────────┘

3. حدّث المتصفح
   ┌───────────────────────────────┐
   │ Android: Chrome 89+           │
   │ iOS: Safari 16.4+             │
   │ Desktop: Chrome/Edge/Safari   │
   └───────────────────────────────┘

🔋 استهلاك البطارية:

   شاشة مطفأة + صوت      ~3-5% / ساعة   ⚡

   تشغيل عادي             ~5-8% / ساعة   ⚡
